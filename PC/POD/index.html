About

This is a racing game. The main prize is a ticket to escape the Planet of Death.

Music

The game has CD Audio music composed by:

  • Daniel Masson
  • Raphael Meroño
  • Gaëlle Gauthier
  • Gilles Gries

There is only a single long track.

Graphics

Featured version uses 3DFX card providing 640x480 resolution with 64k colors.

Version specifics

The featured release is called POD 2.0, which is a base release with 3D accelerator support.

Others

This game can be also called POD: Planet of Death.

An expansion pack is available but is does not introduce any new music.
