About

Lemmings are tiny creatures that march into each level and must be guided safely to the exit by assigning them special abilities in real time.

Music

The CD-ROM release has CD Audio tracks composed by:

  • Brian Johnston
  • Tim Wright
  • David Whittaker

Four tracks reference other Psygnosis games. Half of the remaining are public domain compositions and the other half is own music composed by Brian Johnston and Tim Wright. See Subtitle field in track list. For even more details see Music in Lemmings wiki

The original floppy release has OPL3 music available at gamus.space.

Graphics

The game uses 320x200 resolution with 16 colors. Menus use resolution of 640x350.

Version specifics

The original floppy version was released in 1991.

PC versions do not have the introduction sequence that Amiga has.

CD-ROM release replaced OPL3 soundtrack with CD Audio tracks.

The CD-ROM release also contains the expansion pack Oh No! More Lemmings but it has no rearranged expansion music so it plays the original soundtrack.
