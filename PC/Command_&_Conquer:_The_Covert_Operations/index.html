About

This is an expansion pack for Command & Conquer: Tiberian Dawn which contains several new missions.

Music

The music is composed by:

  • Frank Klepacki
  • Rod Arnett
  • Dan Ryan
  • Greg Greer

The expansion adds a noticeable amount of new tracks.

As a bonus there are CD Audio versions of a few tracks.

Graphics

The game uses resolution of 320x200 pixels with 256 colors.

Version specifics

The PC version is the reference version.
