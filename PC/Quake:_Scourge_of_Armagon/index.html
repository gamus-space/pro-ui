About

The first expansion pack to the 3D first-perspective shooter game. The hero crawls more dungeons to face Armagon.

Expansion features new weapons, enemy types and power-ups located in 18 levels divided in 3 episodes.

Music

Game has CD Audio tracks composed by (according to game manual):

  • Jeehun Hwang

Graphics

Featured version uses 3DFX card providing 640x480 resolution with 64k colors.

Version specifics

The game was originally released for DOS with software rendering support. A year later an mini-OpenGL port was released for Windows 95, called glQuake.

Others

Featured version: glQuake 0.98 (Quake 1.09)
