About

A sequel to Screamer 2, now redesigned in an arcade‑style rally format with 5 cars and 7 international tracks.

Music

The game has a CD Audio soundtrack composed by:

  • Christian Björklund
  • Lee Nicklen

Graphics

The featured 3DFX version uses 640x480 resolution with 64k colors.

While playing the League, tracks can randomly appear with foggy or nightly conditions.

Note that this versions displays some artifacts on floor tiles connections.

Others

Featured version: 3DFX

This game is also known as Bleifuss Rally.
