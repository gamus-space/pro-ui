About

This is the first expansion pack for Red Alert which contains several new missions.

Music

The music is composed by:

  • Frank Klepacki

The expansion adds some new side-specific tracks.

As a bonus there are CD Audio versions of a few tracks.

Graphics

The DOS version uses resolution of 320x200 pixels with 256 colors.

The featured Windows version is upgraded to 640x400 resolution with 256 colors.

Version specifics

The PC version is the reference version.
