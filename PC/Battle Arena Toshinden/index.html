About

One of the first 3D fighting games. Eight fighters meet at the Battle Arena Toshinden tournament. They are:

  • Eiji Shinjo
  • Kayin Amoh
  • Sofia
  • Rungo Iron
  • Fo Fai
  • Mondo
  • Duke B. Rambert
  • Ellis

Music

Game has CD Audio tracks composed by (according to PSX version credits):

  • Yasuhiro Nakano
  • Makoto Mukai
  • Fumio Tanabe

The game also features MIDI music arranged by:

  • The Fatman

OPL3 version is available at gamus.space.

Graphics

Basic version supports software rendering.

Featured version has 3DFX patch, allowing the game to run smoothly on 640x480 resolution with 64k colors.

Version specifics

The PC version has an extra hidden boss: Earthworm Jim!
