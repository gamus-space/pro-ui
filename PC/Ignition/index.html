About

Dynamic small cars racing game with tracks filled with numerous hazards.

Features:

  • 7 cars + 4 unlockable
  • 5 tracks + 2 unlockable
  • 4 championship levels

Music

Graphics

The game was released with software rendering support.

Featured version has a 3DFX patch that drives on 640x400 resolution with 64k colors.

Version specifics

This game was released only to PC.
