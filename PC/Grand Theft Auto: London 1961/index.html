About

This is the second expansion pack to Grand Theft Auto.

Like the first expansion pack, the game is also set in London, taking place eight years earlier. It features new vehicles and additional missions.

Music

The game has CD Audio soundtrack researched and compiled by:

  • Patrick Whittaker
  • Terry Donovan

Graphics

Game supports variety of resolutions in 256 or 64k colors. The featured 3DFX version displays 640x480 with 64k colors.

Version specifics

The PC version is the reference version.

Others

Featured version: 11.5 (European release)

This game is actually called Grand Theft Auto Mission Pack #2: London 1961.

This game was released as free to download and required the first expansion pack.
