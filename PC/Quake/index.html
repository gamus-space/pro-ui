Platform PC
Year 1996
Developer id Software

System DOS/Windows 95
Date added 2024-08-12

About

A mainstream 3D first-perspective shooter game. The hero crawls medieval dungeons to face Shub-Niggurath.

Game features 8 weapons and several enemy types located in 30 levels divided in 4 episodes.

Music

Game has CD Audio tracks composed by:

  • Trent Reznor and Nine Inch Nails

The soundtrack is ambient.

Graphics

Featured version uses 3DFX card providing 640x480 resolution with 64k colors.

Version specifics

The game was originally released for DOS with software rendering support. A year later an mini-OpenGL port was released for Windows 95, called glQuake.
