About

The game is the final upgrade to Mortal Kombat 3. It adds up characters and backgrounds from the first and second tournament, as well as some new characters.

Music

Game has CD Audio music was composed by:

  • Dan Forden

Compared to the previous editions of Mortal Kombat 3 the music plays significantly slower.

Graphics

The game uses resolution of 320x240 pixels with 64k colors.

Version specifics

The PC version is a port of the PlayStation version.

Others

The game has DOS and Windows versions which seem to be identical.
