About

This is an expansion pack for Command & Conquer: Tiberian Sun which contains a new campaign for both sides.

Music

The music is composed by:

  • Frank Klepacki
  • Jarrid Mendelson

The new music is the same for both sides.

The game features a jukebox, allowing to select track during gameplay.

The music is stored in AUD proprietary format. It has quality of 22kHz 16bit mono and uses ADPCM compression.

Graphics

The game allows to pick a resolution of 640x400, 640x480 (featured) or 800x600 pixels with 64k colors.
