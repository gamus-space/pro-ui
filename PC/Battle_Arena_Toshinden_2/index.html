About

A sequel of a 3D fighting game. A year has passed after the first tournament. The new competitors join:

  • Gaia (former boss)
  • Tracy
  • Chaos

Music

Game has CD Audio-like tracks composed by:

  • Yasuhiro Nakano
  • Fumio Tanabe

The music is stored in uncompressed 22kHz 8bit stereo files.

Graphics

The game supports Direct3D, allowing to run with variety of resolutions. Screenshots use 640x480 with 64k colors.

Version specifics

The PC version feels like a PlayStation emulator, so the game is almost identical to the base version.
