Platform PC
Year 1997
Developer Synergistic Software
Publisher Sierra

Date added 2024-06-01

About

The is an expansion pack for Diablo featuring:

  • new character classes
  • new locations
  • new monsters
  • new items
  • new spells

The goal is to eliminate Na-Krul that was accidentally released by a sorcerer.

Music

The game has the soundtrack in packaged 22kHz 16bit stereo WAV files composed by:

  • Matt Uelmen
  • Craig Utterback

It is not clear who the author is. The original composer appears is WAV files metadata, but he's not credited in the game.

Graphics

The game uses resolution of 640x480 pixels with 256 colors.

Version specifics

The PC version was the base release.
