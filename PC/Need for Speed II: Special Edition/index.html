About

This is an extended version of Need for Speed II racing game. It features more cars, a new track and 3DFX accelerator support.

Music

The music is composed by:

  • Saki Kaskas
  • Jeff van Dyck
  • Alistair Hirst

Music is stored in proprietary MAP/MUS and ASF files with 22kHz 16bit stereo quality.

The game is capable to change music based on race conditions.

Graphics

Featured version uses 3DFX card providing 640x480 resolution with 64k colors.
