About

Tomb Raider is one of earliest 3D third-person action/adventure games. An archeologist called Lara Croft is given a quest to travel to Peru in order to collect an ancient artifact.

Music

Game has CD Audio tracks composed by:

  • Nathan McCree

Graphics

Basic version supports software rendering.

Featured version has 3DFX patch, making the game run smoothly on 640x480 resolution with 64k colors.

Version specifics

Most notable difference is the lack of save crystals, thus allowing the player to save everywhere.

The PC version does not contain the contextual music that console versions have.

Others

Featured version: European release with 3DFX patch (Feb 11 1997)

After the original release, a few patches were prepared to support the mainstream 3D hardware accelerators.

ATI patched version is the only one that requires Windows 95. It is also the most advanced one.
