About

This is sequel of a mainstream racing game. It features 8 licensed sports cars.

Music

The music is composed by:

  • Saki Kaskas
  • Jeff van Dyck
  • Alistair Hirst

Music is stored in proprietary MAP/MUS and ASF files with 22kHz 16bit stereo quality.

The game is capable to change music based on race conditions.

Graphics

The game uses 640x480 resolution with 256 colors. The menus use 64k colors.
