About

This is a 2D fighting game. It is the first Capcom game using characters from Marvel comics.

Music

The game has CD Audio music composed by:

  • Takayuki Iwai
  • Shun Nishigaki
  • Hideki Okugawa
  • Isao Abe

Graphics

The game uses a resolution of 640x480 pixels with 256 colors (despite the claim to use 32k colors).
