About

The expansion pack contains additional 2 Egyptian levels and 2 new Atlantean levels. There is not much storyline available for the new mini-quest.

Music

Game has CD Audio tracks composed by:

  • Nathan McCree

The expansion soundtrack is a subset of the original soundtrack.

Graphics

Basic version supports software rendering.

Featured version has 3DFX patch, allowing the game to run smoothly on 640x480 resolution with 64k colors.

Version specifics

The expansion pack was released to PC only.

Others

Featured version: European release with 3DFX patch (Feb 16 1998)

Like the original, this game also received some 3D hardware accelerators patches.
