About

This is a vehicular combat racing game.

The races can be won by either:

  • finishing all laps
  • destroying all enemy cars
  • running over all zombies

Music

Game has CD Audio tracks composed by (according to game manual):

  • Lee Groves
  • Fear Factory

There is composer information for individual tracks available on the Internet.

The game also contains MIDI music which is not played in the final version. However the OPL3 version can be listened to at gamus.space.

Graphics

The featured 3DFX version uses 640x480 resolution with 64k colors.

Game features 36 races but only 11 distinct locations are presented.

The featured version is censored and contains zombies as pedestrians.

Others

Featured version: 3DFX patch.

The original aim of the game was to run over pedestrians. This was censored by replacing people with zombies.
