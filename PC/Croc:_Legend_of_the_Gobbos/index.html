Croc: Legend of the Gobbos
Platform PC
Year 1997
Developer Argonaut Software
Publisher Fox Interactive
System Windows 95
Date added 2023-11-25

About

It's a 3D third-perspective platform game. The young crocodile Croc must rescue his fellow Gobbos friends from the evil Baron Dante.

Music

Game has CD Audio tracks composed by:

  • Karin Griffin
  • Martin Gwynn Jones
  • Justin Scharvona

Graphics

The game uses Windows version of BRender engine, which supports a few 3D accelerators natively.

The featured version uses 3DFX with 640x480 resolution and 64k colors.

Version specifics

The PC version features about 66% of music included the PlayStation version.

Others

Featured version: 1.0 (USA release)
