About

This is an expansion pack for Descent II containing 20 regular and 3 secret levels.

Music

Game has CD Audio tracks composed by:

  • Brian Luzietti
  • Ogre of Skinny Puppy, Mark Walk
  • Type O Negative
  • Mark Morgan
  • Larry Peacock

There is composer information for individual tracks available on the Internet.

Unfortunately when running the expansion missions the game plays the original track set, because it fails to start with the expansion CD inserted into drive. To work around this issue it is assumed that expansion missions play the expansion music sequentially (like the original game).

The game also contains MIDI music and the OPL3 version can be listened to at gamus.space.

Graphics

The featured 3DFX version uses 640x480 resolution with 64k colors.
