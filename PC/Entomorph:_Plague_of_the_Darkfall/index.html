About

An action/adventure/RPG game. After an encounter with mysterious spider, squire Warrick wakes up on Hamis island, trying to find his sister.

Music

Game has CD Audio tracks composed by:

  • Rick Rhodes
  • Danny Pelfrey

Graphics

Game uses 640x480 resolution with 256 colors.

Others

Featured version: 1.0

This game is the second one (after World of Aden: Thunderscape) based in the World of Aden universe.

As an early Windows game, it was supposed to run on Windows 3.11 with Win32s, which could not be confirmed. However it runs perfectly on Windows 95.
