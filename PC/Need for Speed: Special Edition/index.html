Platform PC
Year 1996
Developer Electronic Arts Canada
Publisher Electronic Arts
System DOS/Windows 95
Date added 2024-04-12

About

This is an extended version of Need for Speed racing game that features 8 licensed sports cars.

Music

The music is composed by:

  • Jeff Dyck
  • Saki Kaskas
  • Edwin Dolinski

The same music set is used in the menu and during racing.

Music is stored in proprietary ASF/AS4 files with 16/22kHz 16bit stereo quality.

Graphics

The game uses 640x480 resolution with 256 colors.

Version specifics

The Special Edition is the last and most advanced version.
