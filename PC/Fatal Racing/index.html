About

A racing game containing 8 teams. Every team provides a track for each of 3 cups (24 tracks in total) and a unique car.

Music

The game has a CD Audio soundtrack composed by:

  • Neil Biggin

Alternatively MIDI versions of the tracks can be played.

The OPL3 version is available at gamus.space.

Graphics

The 3DFX version has 640x480 resolution with 64k colors.

Version specifics

The 3DFX version has some drawbacks comparing to original version:

  • no loading screen with game title
  • no CD Audio support
  • missing statistics on game screen
  • no fireworks on championship ending

Others

Featured version: 3DFX patch

This game is also known as Whiplash (in North America).
