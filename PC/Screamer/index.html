About

A racing game containing 6 cars and 6 tracks.

Music

The game has a CD Audio soundtrack composed by:

  • Allister Brimble
  • John Symonds
  • Neal Reynolds

Alternatively MIDI versions of the tracks can be played.

The OPL3 version is available at gamus.space.

Graphics

The game uses 320x200 or 640x480 (featured) resolution with 256 colors.

Others

Featured version: European release

This game is also known as Bleifuss.
