About

This is a 3D shooting game. The goal is to capture the flag of either static CPU opponent, or compete with another player in split screen mode.

The available vehicles are:

  • ASV
  • Tank
  • Helicopter
  • Jeep

Music

The game features classical music stored in CD quality single huge uncompressed WAV file.

Graphics

Only software rendering is supported in either 320x200 or 640x480 (featured) resolution with 256 colors.
