About

This is an expansion pack for Red Alert 2. Contains new units, buildings, and campaigns. Allows to play as Yuri's side but only in multiplayer/skirmish mode.

Music

The music is composed by:

  • Frank Klepacki

The music is the same for both sides.

The game features a jukebox, allowing to select track during gameplay.

The music is stored in ADPCM-compressed WAV files. It has quality of 22kHz 16bit stereo.

Graphics

The game allows to pick a resolution of 640x480 (featured), 800x600 or 1024x768 pixels with 64k colors.
