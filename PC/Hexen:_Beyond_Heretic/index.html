About

This first-perspective fantasy shooter was one of the mainstream games using Doom engine. It is a sequel to Heretic.

Game consists of 31 levels divided in 5 units, which can be wiped out by different character classes:

  • Fighter
  • Cleric
  • Mage

Music

Game has CD Audio tracks composed by:

  • Kevin Schilder

The game can play CD Audio tracks or MIDI music.

The OPL3 version is available online at gamus.space.

Graphics

Windows version allows to pick resolution of 320x200, 320x240, 640x400 or 640x480 (featured) with 256 colors.

The game was originally released for DOS running on 320x200 resolution.

Version specifics

The Windows version was released in 1997.

Others

Featured version: 1.1.

There was an expansion pack Deathkings of the Dark Citadel released in 1996, but it does not contain any new music (neither CD Audio nor MIDI).
