About

This real-time strategy game is a prequel to Command & Conquer: Tiberian Dawn. Albert Einstein travels back in time to eliminate Adolf Hitler. As a consequence an alternate World War II breaks out between the Allies and the Soviet.

Music

The music is composed by:

  • Frank Klepacki

Some tracks are common for both sides and others are side-specific.

The game features a jukebox, allowing to select track during gameplay. For some reason majority of Allied missions start with "Bigfoot" track, which is unavailable for Allies in the jukebox and so cannot be played manually.

The music is stored in AUD proprietary format. It has quality of 22kHz 16bit mono and uses ADPCM compression.

Graphics

The DOS version uses resolution of 320x200 pixels with 256 colors.

The featured Windows version is upgraded to 640x400 resolution with 256 colors.

Version specifics

The PC version is the reference version.
