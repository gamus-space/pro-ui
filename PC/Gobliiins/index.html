Graphics

The game uses 320x200 resolution with 256 colors.

The CD-ROM release has improved graphics compared to the original.

Version specifics

The original floppy version was released in 1991.

As mentioned before, CD-ROM version added music and improved the graphics. Because of added digitized voices, all texts were removed from the game.
