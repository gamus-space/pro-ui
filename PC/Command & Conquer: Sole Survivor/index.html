About

This is a multiplayer-only action game based on Command & Conquer: Tiberian Dawn.

Each player controls a single unit in a free-for-all arena shooter style game, with extra power-ups available in crates.

Music

The music is composed by:

  • Frank Klepacki

Soundtrack is a compilation of:

  • single track from Command & Conquer: Tiberian Dawn
  • 4 tracks from Command & Conquer: The Covert Operations
  • 4 tracks from Red Alert

Each track has its additional CD Audio version.

Graphics

The game uses resolution of 640x480 pixels with 256 colors.
